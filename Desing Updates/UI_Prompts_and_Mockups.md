# CommonGround UI Mockups & Visual Prompts

This document contains the visual design specifications, specific image generation prompts, and the resulting high-fidelity mockups for the core screens of the CommonGround Operating System.

## 1. Global Aesthetic Strategy
*   **Vibe:** Organic Minimalist ("The Sanctuary of Truth")
*   **Palette:** Sage Green (`#4A6C58`), Slate Blue (`#475569`), Warm Sand Backgrounds.
*   **Typography:** Clean Sans-Serif (Inter/Roboto) for UI, Serif (Merriweather) for Agreements.
*   **Iconography:** Font Awesome / Vector line icons (Clean, thin stroke).
