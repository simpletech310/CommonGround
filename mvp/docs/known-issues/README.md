# Known Issues Index

This directory tracks all known issues in the project.

## Active Issues
(Issues are automatically listed here by the log_error.py script)

## How to Use
1. Run `python scripts/log_error.py` when you encounter an error
2. The script creates a new issue file here
3. Update the issue file as you investigate
4. Move to SOLVED/ when fixed

## Issue File Format
Files are named: `XXX-brief-description.md`
- XXX = Issue number (auto-incremented)
- brief-description = Short slug of the issue
